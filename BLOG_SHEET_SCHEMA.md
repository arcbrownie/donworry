## 사용 팁

### 1. 필터링 활용
- 카테고리별 필터로 카테고리별 관리
- 상태별 필터로 발행/초안 구분
- 구글 색인 여부 필터로 색인 상태 확인

### 2. 데이터 검증
- URL 슬러그는 소문자, 하이픈(-)만 사용
- 발행일/수정일은 YYYY-MM-DD 형식 준수
- 메타 설명은 150-160자 이내 권장

### 3. 자동화 연동
- 구글 색인 여부는 구글 서치콘솔 API로 자동 업데이트 가능
- 조회수/체류시간은 구글 애널리틱스 API로 자동 연동 가능
