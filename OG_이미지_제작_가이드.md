## OG 이미지란?

**OG 이미지(Open Graph Image)**는 소셜 미디어에서 링크를 공유할 때 표시되는 미리보기 이미지입니다.

### 언제 보이나요?
- 카카오톡에서 링크 공유 시
- 페이스북/인스타그램에서 링크 공유 시
- 트위터/X에서 링크 공유 시
- 슬랙, 디스코드 등 메신저에서 링크 공유 시

### 왜 중요한가요?
- **클릭률 향상**: 예쁜 이미지가 있으면 더 많이 클릭됩니다
- **브랜드 인지도**: 일관된 디자인으로 브랜드 인지도 상승
- **SEO 효과**: 소셜 공유가 많아지면 검색 노출도 증가

## 브랜드 정보 정리

### 로고
- **파일**: `public/favicon.svg`
- **디자인**: 방패 아이콘 (Shield)
- **그라디언트**: Deep Blue → Deep Purple
- **색상 코드**:
  - 시작: `hsl(232, 68%, 35%)` (Deep Blue)
  - 끝: `hsl(280, 78%, 40%)` (Deep Purple)

### 브랜드 컬러
- **메인 컬러 (Primary)**: 
  - Deep Blue: `#1A237E` 또는 `hsl(232, 68%, 30%)`
  - RGB: `(26, 35, 126)`
  
- **보조 컬러 (Secondary)**:
  - Deep Purple: `#4A148C` 또는 `hsl(280, 78%, 31%)`
  - RGB: `(74, 20, 140)`

### 폰트
- **한글**: Noto Sans KR
- **영문**: Sans-serif (시스템 기본)

### 슬로건
- **메인**: "일상은 가볍게, 돈 걱정은 없게"
- **서브**: "어제보다 가벼운 오늘을 만드는 당신의 솔루션"

### 서비스 설명
- 금융 솔루션 플랫폼
- 계산기, 가이드, 채무조정 진단 제공

## 제작 방법 (단계별)

### 방법 1: Canva 사용 (추천 - 가장 쉬움)

#### 1단계: Canva 계정 만들기
1. https://www.canva.com/ 접속
2. 무료 계정 생성 (이메일 또는 구글 계정)

#### 2단계: 템플릿 선택
1. "디자인 만들기" 클릭
2. 크기 입력: **1200 x 630** (픽셀)
   - 또는 "Facebook Post" 템플릿 선택 후 크기 변경
3. "만들기" 클릭

#### 3단계: 배경 설정
1. 왼쪽 메뉴에서 "배경" 선택
2. 그라디언트 배경 선택 또는 직접 만들기:
   - **옵션 A**: 그라디언트 배경 템플릿 사용
   - **옵션 B**: 직접 만들기
     - 색상 1: `#1A237E` (Deep Blue)
     - 색상 2: `#4A148C` (Deep Purple)
     - 방향: 대각선 (왼쪽 위 → 오른쪽 아래)

#### 4단계: 로고 추가
1. 왼쪽 메뉴에서 "업로드" 클릭
2. `public/favicon.svg` 파일 업로드
   - SVG가 안 되면 PNG로 변환 후 업로드
3. 로고를 왼쪽 상단 또는 중앙 상단에 배치
4. 크기 조정: 너무 크지 않게 (전체 높이의 15-20% 정도)

#### 5단계: 텍스트 추가
1. 왼쪽 메뉴에서 "텍스트" 선택
2. **제목 텍스트 추가**:
   - 내용: "돈워리"
   - 폰트: 굵은 폰트 (Bold)
   - 크기: 80-100px
   - 색상: 흰색 (#FFFFFF)
   - 위치: 로고 옆 또는 중앙

3. **슬로건 텍스트 추가**:
   - 내용: "일상은 가볍게, 돈 걱정은 없게"
   - 폰트: 중간 굵기 (Medium)
   - 크기: 40-50px
   - 색상: 흰색 또는 연한 회색 (#F5F5F5)
   - 위치: 제목 아래

4. **서브 텍스트 추가** (선택):
   - 내용: "어제보다 가벼운 오늘을 만드는 당신의 솔루션"
   - 폰트: 일반 (Regular)
   - 크기: 28-32px
   - 색상: 연한 흰색 (#E0E0E0)
   - 위치: 슬로건 아래

#### 6단계: 아이콘/요소 추가 (선택)
1. 왼쪽 메뉴에서 "요소" 또는 "아이콘" 선택
2. 간단한 아이콘 추가:
   - 💰 계산기
   - 📊 가이드
   - 🛡️ 채무조정
3. 작고 은은하게 배치 (너무 강조하지 않기)

#### 7단계: 다운로드
1. 우측 상단 "다운로드" 버튼 클릭
2. 파일 형식: **PNG** 선택
3. 품질: "높음" 또는 "최고" 선택
4. 다운로드

#### 8단계: 파일 이름 변경 및 이동
1. 다운로드한 파일 이름을 `og-image.png`로 변경
2. 프로젝트의 `public/` 폴더에 복사

### 방법 2: Figma 사용 (더 전문적)

#### 1단계: Figma 계정 만들기
1. https://www.figma.com/ 접속
2. 무료 계정 생성

#### 2단계: 새 파일 생성
1. "New design file" 클릭
2. 프레임 크기 설정: **1200 x 630**

#### 3단계: 배경 레이어
1. Rectangle 도구로 전체 영역 선택
2. Fill 설정:
   - Type: **Linear Gradient**
   - 색상 1: `#1A237E` (위쪽)
   - 색상 2: `#4A148C` (아래쪽)
   - 각도: 135도

#### 4단계: 로고 추가
1. File → Import로 `favicon.svg` 가져오기
2. 크기 조정 및 위치 설정

#### 5단계: 텍스트 추가
1. Text 도구로 텍스트 추가
2. 폰트: Noto Sans KR (없으면 시스템 폰트 사용)
3. 위와 동일한 텍스트 구성

#### 6단계: Export
1. 프레임 선택
2. 우측 패널에서 Export 설정
3. Format: PNG, Scale: 1x
4. Export 클릭


### 색상 팔레트

#### 메인 그라디언트
```
시작: #1A237E (Deep Blue)
  ↓
끝:   #4A148C (Deep Purple)
```

#### 텍스트 색상
- **제목**: #FFFFFF (흰색)
- **슬로건**: #F5F5F5 (연한 흰색)
- **서브 텍스트**: #E0E0E0 (더 연한 흰색)

### 폰트 크기 가이드 (1200x630 기준)
- **브랜드명 (돈워리)**: 80-100px
- **슬로건**: 40-50px
- **서브 텍스트**: 28-32px
- **아이콘/요소**: 24-32px

### 여백 가이드
- **좌우 여백**: 60-80px
- **상하 여백**: 40-60px
- **요소 간 간격**: 20-30px

### 주의사항
1. **텍스트 가독성**: 배경이 어두우니 텍스트는 밝게
2. **중요 정보**: 브랜드명과 슬로건은 명확하게
3. **과하지 않게**: 너무 많은 요소는 피하기
4. **모바일 고려**: 작은 화면에서도 잘 보이게

## 최종 확인 및 테스트

### 1. 파일 확인
- [ ] 파일명: `og-image.png`
- [ ] 위치: `public/og-image.png`
- [ ] 크기: 1200 x 630 픽셀
- [ ] 용량: 1MB 이하 (권장)

### 2. 메타 태그 확인
`index.html`에 이미 설정되어 있는지 확인:
```html
<meta property="og:image" content="https://donworry.kr/og-image.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
```

### 3. 배포 후 테스트

#### Facebook Sharing Debugger
1. https://developers.facebook.com/tools/debug/ 접속
2. URL 입력: `https://donworry.kr`
3. "디버그" 클릭
4. 이미지가 제대로 표시되는지 확인
5. 문제가 있으면 "스크래핑 다시 실행" 클릭

#### Twitter Card Validator
1. https://cards-dev.twitter.com/validator 접속
2. URL 입력: `https://donworry.kr`
3. "Preview card" 클릭
4. 이미지 확인

#### 카카오톡 링크 미리보기
1. 카카오톡에서 `https://donworry.kr` 링크 입력
2. 미리보기 확인
3. 이미지가 표시되는지 확인

### 4. 문제 해결

#### 이미지가 안 보일 때
1. **캐시 문제**: 
   - Facebook Debugger에서 "스크래핑 다시 실행"
   - 브라우저 캐시 삭제
   
2. **파일 경로 확인**:
   - `public/og-image.png` 파일이 실제로 존재하는지
   - 배포 후 `https://donworry.kr/og-image.png`로 직접 접속 가능한지

3. **파일 형식 확인**:
   - PNG 또는 JPG 형식인지
   - 파일명이 정확한지 (대소문자 구분)

4. **메타 태그 확인**:
   - `index.html`에 올바른 경로가 설정되어 있는지
   - 절대 경로 사용 (`https://donworry.kr/og-image.png`)

## 추가 팁

### 여러 페이지용 OG 이미지 (고급)
각 페이지마다 다른 OG 이미지를 만들 수도 있습니다:
- 메인 페이지: 일반 브랜드 이미지
- 블로그 글: 글 제목이 포함된 이미지
- 계산기 페이지: 계산기 관련 이미지

이 경우 각 페이지의 `useEffect`에서 동적으로 설정:
```typescript
useEffect(() => {
  const ogImage = document.querySelector('meta[property="og:image"]');
  if (ogImage) {
    ogImage.setAttribute('content', 'https://donworry.kr/og-image-blog.png');
  }
}, []);
```

### 이미지 최적화
- **TinyPNG**: https://tinypng.com/ - 이미지 압축
- **Squoosh**: https://squoosh.app/ - 이미지 최적화 도구
