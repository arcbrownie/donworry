## 🚀 빠른 시작

### 새로운 블로그 포스트 작성하기

1. [블로그 스타일 및 작성 가이드](./블로그_스타일_및_작성_가이드.md)를 참고하세요
2. `src/pages/content/` 폴더에 새 컴포넌트 파일 생성
3. 전문가를 선택하고 `ArticleAuthor` 컴포넌트 추가
4. `src/lib/experts.ts`의 `expertArticles`에 관련 글 등록

### 성능 최적화

- [성능 최적화 가이드](./성능_최적화_가이드.md)를 따라 최적화 작업 수행
- [Lighthouse 오류 해결 가이드](./Lighthouse_오류_해결_가이드.md)로 점수 개선
