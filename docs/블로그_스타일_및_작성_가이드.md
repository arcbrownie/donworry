# 블로그 스타일 및 작성 가이드

돈워리 블로그 포스트 작성 시 따라야 할 스타일 가이드와 구조입니다.

## 📁 파일 구조

모든 블로그 포스트는 `src/pages/content/` 디렉토리에 위치합니다.

```
src/pages/content/
  ├── SocialBeginnerSalaryGuide.tsx
  ├── FirstLoanGuide2030.tsx
  ├── InvestmentGuideForBeginners.tsx
  └── ...
```

## 🎨 기본 구조

모든 블로그 포스트는 다음 구조를 따라야 합니다:

```tsx
import { useEffect, useState } from "react";
import { Link, useLocation } from "react-router-dom";
import { setCanonicalTag, addStructuredData, createExpertArticleSchema } from "@/lib/utils";
import MainNavigation from "@/components/layout/MainNavigation";
import Footer from "@/components/layout/Footer";
import MobileFloatingCTA from "@/components/ui/MobileFloatingCTA";
import TableOfContents from "@/components/ui/TableOfContents";
import { ArticleAuthor } from "@/components/ui/ArticleAuthor";
// 필요한 다른 컴포넌트들...

export default function YourArticleName() {
  const location = useLocation();
  const [scrollProgress, setScrollProgress] = useState(0);

  // 스크롤 진행도 계산
  useEffect(() => {
    const handleScroll = () => {
      const windowHeight = window.innerHeight;
      const documentHeight = document.documentElement.scrollHeight;
      const scrollTop = window.scrollY;
      const progress = (scrollTop / (documentHeight - windowHeight)) * 100;
      setScrollProgress(Math.min(100, Math.max(0, progress)));
    };

    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, []);

  // SEO 설정
  useEffect(() => {
    setCanonicalTag(location.pathname);
    document.title = "글 제목 | 돈워리";
    
    // 메타 설명 설정
    let metaDescription = document.querySelector('meta[name="description"]');
    if (!metaDescription) {
      metaDescription = document.createElement('meta');
      metaDescription.setAttribute('name', 'description');
      document.head.appendChild(metaDescription);
    }
    metaDescription.setAttribute('content', '글의 간단한 설명 (150-160자)');

    // 구조화된 데이터 추가
    const schema = createExpertArticleSchema({
      title: "글 제목",
      description: "글 설명",
      expertId: "expert-savings-01", // 전문가 ID
      publishedDate: "2026-01-27",
      lastUpdated: "2026-01-27",
    });
    addStructuredData(schema);
  }, [location.pathname]);

  return (
    <div className="min-h-screen bg-background pb-24 md:pb-0">
      <MainNavigation />

      {/* Sticky Progress Bar */}
      <div className="sticky top-[80px] z-40 w-full h-1 bg-slate-200">
        <div 
          className="h-full bg-gradient-to-r from-[#1A237E] via-[#4A148C] to-[#1A237E] transition-all duration-150"
          style={{ width: `${scrollProgress}%` }}
        />
      </div>

      {/* Hero Section */}
      <section className="relative overflow-hidden bg-gradient-to-br from-[#1A237E] via-[#4A148C] to-[#1A237E] py-12 md:py-16">
        <div className="absolute inset-0 bg-[radial-gradient(ellipse_at_top_right,rgba(255,255,255,0.15)_0%,transparent_50%)]" />
        <div className="absolute -right-10 -bottom-10 w-40 h-40 bg-white/10 rounded-full blur-3xl" />
        <div className="container relative z-10">
          <div className="text-center text-white">
            <h1 className="text-2xl md:text-3xl lg:text-4xl font-extrabold mb-4 leading-tight">
              글의 메인 제목
            </h1>
            <p className="text-lg md:text-xl text-white/90 max-w-2xl mx-auto">
              글의 부제목 또는 간단한 설명
            </p>
          </div>
        </div>
      </section>

      {/* Main Content */}
      <main className="container py-12">
        <div className="flex flex-col lg:flex-row gap-8">
          {/* Table of Contents */}
          <div className="lg:w-64 flex-shrink-0">
            <TableOfContents />
          </div>

          {/* Article Content */}
          <article className="flex-1 max-w-4xl prose prose-lg prose-slate max-w-none">
            {/* Introduction */}
            <section className="mb-10">
              <p className="text-muted-foreground leading-relaxed text-base mb-4">
                글의 도입부 내용...
              </p>
            </section>

            {/* Sections */}
            <section className="mb-10">
              <h2 id="section-id" className="text-2xl font-bold text-foreground mb-4 flex items-center gap-3 scroll-mt-20">
                <Icon className="w-7 h-7 text-[#1A237E]" />
                섹션 제목
              </h2>
              <p className="text-muted-foreground leading-relaxed mb-4">
                섹션 내용...
              </p>
            </section>

            {/* Article Author */}
            <ArticleAuthor 
              expertId="expert-savings-01"
              publishedDate="2026-01-27"
              lastUpdated="2026-01-27"
            />
          </article>
        </div>
      </main>

      <MobileFloatingCTA />
      <Footer />
    </div>
  );
}
```

## 👤 전문가 배치 가이드

### 1. 전문가 선택

각 블로그 포스트는 반드시 한 명의 전문가를 배치해야 합니다. 전문가 선택 기준:

- **재테크 · 절약 카테고리**: `expert-savings-01`, `expert-savings-02`, `expert-savings-03`
- **금융 · 대출 카테고리**: `expert-finance-01`, `expert-finance-02`, `expert-finance-03`
- **채무조정 카테고리**: `expert-debt-01`, `expert-debt-02`, `expert-debt-03`

### 2. 전문가 정보 확인

`src/lib/experts.ts` 파일에서 전문가 정보를 확인할 수 있습니다:

```typescript
// 재테크 · 절약 전문가
- expert-savings-01: 김민지 (재테크 전문가)
- expert-savings-02: 이수진 (절약 전문가)
- expert-savings-03: 박지훈 (투자 전문가)

// 금융 · 대출 전문가
- expert-finance-01: 최동현 (대출 전문가)
- expert-finance-02: 정서연 (금리 전문가)
- expert-finance-03: 강민수 (세무 전문가)

// 채무조정 전문가
- expert-debt-01: 윤서진 (개인회생 전문가)
- expert-debt-02: 한지우 (채무조정 전문가)
- expert-debt-03: 서준호 (신용회복 전문가)
```

### 3. ArticleAuthor 컴포넌트 사용

글 하단에 전문가 정보를 표시하려면 `ArticleAuthor` 컴포넌트를 사용합니다:

```tsx
<ArticleAuthor 
  expertId="expert-savings-01"  // 전문가 ID
  publishedDate="2026-01-27"     // 작성일 (선택)
  lastUpdated="2026-01-27"       // 최종 수정일 (선택)
/>
```

### 4. 전문가별 관련 글 매핑

`src/lib/experts.ts`의 `expertArticles` 객체에 전문가별 관련 글을 등록합니다:

```typescript
export const expertArticles: Record<string, string[]> = {
  'expert-savings-01': ['/content/social-beginner-salary-guide', '/content/one-person-household-savings'],
  'expert-savings-02': ['/content/one-person-household-savings'],
  // ...
};
```

## 📑 목차 생성 가이드

### 1. 자동 목차 생성

`TableOfContents` 컴포넌트는 자동으로 `<h2>` 태그를 감지하여 목차를 생성합니다.

### 2. 섹션 ID 설정

각 섹션의 `<h2>` 태그에 고유한 `id`를 설정해야 합니다:

```tsx
<h2 id="section-id" className="text-2xl font-bold text-foreground mb-4 flex items-center gap-3 scroll-mt-20">
  섹션 제목
</h2>
```

**중요**: `scroll-mt-20` 클래스를 반드시 추가하여 스크롤 시 헤더에 가려지지 않도록 합니다.

### 3. 목차 스타일

- 데스크톱: 왼쪽 사이드바에 고정(sticky) 표시
- 모바일: 접을 수 있는 박스 형태로 표시
- 현재 섹션은 자동으로 하이라이트됩니다

## 🎨 스타일 가이드

### 색상 팔레트

- **Primary**: `#1A237E` (진한 파란색)
- **Secondary**: `#4A148C` (보라색)
- **Text**: `text-foreground` (기본 텍스트)
- **Muted**: `text-muted-foreground` (보조 텍스트)

### 타이포그래피

- **제목 (H1)**: `text-2xl md:text-3xl lg:text-4xl font-extrabold`
- **섹션 제목 (H2)**: `text-2xl font-bold text-foreground`
- **본문**: `text-muted-foreground leading-relaxed text-base`
- **강조 텍스트**: `<strong className="text-[#1A237E]">텍스트</strong>`

### 아이콘 사용

각 섹션 제목에는 관련 아이콘을 추가합니다:

```tsx
import { Wallet, ShieldCheck, TrendingUp } from "lucide-react";

<h2 className="...">
  <Wallet className="w-7 h-7 text-[#1A237E]" />
  섹션 제목
</h2>
```

### 리스트 스타일

```tsx
<ul className="space-y-3 mb-6">
  <li className="flex items-start gap-3">
    <span className="font-bold text-[#1A237E] text-xl mt-0.5">•</span>
    <div>
      <strong className="text-foreground block mb-1">항목 제목:</strong>
      <span className="text-muted-foreground">항목 설명</span>
    </div>
  </li>
</ul>
```

### 카드/위젯 스타일

중요한 정보나 계산기 위젯은 카드로 감싸서 강조합니다:

```tsx
<Card className="p-6 bg-gradient-to-br from-[#1A237E]/5 via-[#4A148C]/5 to-[#1A237E]/5 border-2 border-[#1A237E]/20 shadow-lg my-8">
  {/* 내용 */}
</Card>
```

## 📝 콘텐츠 작성 가이드

### 1. 도입부

- 독자의 문제 상황을 공감하며 시작
- 글의 핵심 가치 제안을 명확히 제시
- 전문가의 관점에서 접근

### 2. 본문 구조

- 각 섹션은 독립적으로 읽을 수 있도록 구성
- 실용적인 팁과 구체적인 예시 제공
- 숫자와 데이터를 활용한 신뢰성 확보

### 3. 마무리

- 핵심 내용 요약
- 다음 단계 액션 아이템 제시
- 전문가의 조언으로 마무리

## 🎯 전문가별 추천 주제

### 재테크 · 절약 전문가

**김민지 (재테크 전문가)**
- 월급 관리, 예산 설정
- 자산 형성 로드맵
- 복리 저축 전략

**이수진 (절약 전문가)**
- 생활비 절감 솔루션
- 가계부 관리
- 짠테크 노하우

**박지훈 (투자 전문가)**
- 초보자 맞춤형 투자
- 포트폴리오 구성
- 리스크 관리

### 금융 · 대출 전문가

**최동현 (대출 전문가)**
- 대출 상품 비교
- 저금리 대환 대출
- 담보 대출 전략

**정서연 (금리 전문가)**
- 금융상품 수익률 분석
- 신용점수 향상 전략
- 시장 금리 전망

**강민수 (세무 전문가)**
- 연말정산 및 세금 환급
- 청년 세제 혜택
- 소상공인 절세

### 채무조정 전문가

**윤서진 (개인회생 전문가)**
- 개인회생 신청 및 인가
- 파산 절차 안내
- 면책 전략

**한지우 (채무조정 전문가)**
- 채무 조정 프로그램 설계
- 신용회복위원회 연계
- 연체 방지 상담

**서준호 (신용회복 전문가)**
- 신용 회복 후 자산 관리
- 금융 재기 로드맵
- 서민금융 지원
